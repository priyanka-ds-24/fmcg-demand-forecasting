# 📦 FMCG Demand Forecasting Portal

This project is an interactive business intelligence and demand prediction system for FMCG supply chains.

## 🚀 Features
- Sales Trend Analysis
- Demand Forecasting using Linear Regression
- Future Demand Prediction
- Interactive Dashboard using Streamlit
- Business Input Simulation for Demand Prediction

## 🛠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Analylsis of FMCG Dataset
   Capstone Project.ipynb

2. Run the app:
   streamlit run app.py

## 📊 Use Cases
- Inventory Planning
- Supply Chain Optimization
- Business Decision Support
